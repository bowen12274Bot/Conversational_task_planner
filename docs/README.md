# Docs README

本資料夾用於整理專案的文件內容，依照文件用途區分為專案總覽、模組說明、系統設計與後續開發規劃。

## 文件說明

- [project_overview.md](./project_overview.md)
  - 專案主文件
  - 包含專案背景、目標、MVP 範圍、系統核心概念與整體流程

- [module_description.md](./module_description.md)
  - 系統模組說明文件
  - 包含目標輸入模組、AI 引導提問模組、AI 任務拆解模組、規劃面板模組與規劃更新模組

- [system_design.md](./system_design.md)
  - 系統設計文件
  - 包含平台形式、技術架構、技術選型、資料設計方向與 AI 串接設計

- [development_plan](./development_plan)
  - 預留給後續開發計畫、時程安排與任務拆分使用

## 建議閱讀順序

1. `project_overview.md`
2. `module_description.md`
3. `system_design.md`

## 備註

目前文件以 MVP 為主軸整理；若後續擴充完整版本，建議在既有文件中明確標示「MVP」與「後續擴充」的差異。
