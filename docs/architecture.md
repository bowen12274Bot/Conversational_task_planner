## 架構說明

本系統採用三層式架構，以本機 Web 應用形式運作，使用者透過瀏覽器操作。

### 前端介面層（Vue 3 + Vite）

前端負責提供使用者操作介面，包含對話輸入區、訊息顯示區與任務規劃面板。所有與後端的溝通統一透過 `services/api.ts` 發送 HTTP 請求，前端本身不直接呼叫 AI API。

### 後端流程控制層（FastAPI）

後端是本系統的核心，負責接收前端輸入、控制對話流程、呼叫 AI API，並將結果整理後寫入資料庫再回傳前端。AI 串接邏輯封裝於 `services/` 中，與一般路由邏輯分離。

### 資料持久化層（SQLite + SQLAlchemy）

採用 SQLite 作為本機資料庫，透過 SQLAlchemy 管理資料模型與查詢。資料庫檔案存放於 `data/app.db`，不共用、不需額外服務，符合 MVP 本機執行的需求。

### AI 服務（外部 API）

AI 呼叫統一由後端的 service layer 負責，包含組合 prompt、呼叫外部 API、解析回傳結果等工作。此設計使未來更換 AI 模型時，不需大幅修改整體系統。
