# Word README

本資料夾用來存放課程相關文件、老師提供的樣板，以及實際編輯與最終繳交使用的內容。

## 文件分類

### 1. 樣板檔

老師提供的模板，主要作為格式與段落結構參考。

- `W8：期中專題企畫報告樣板(Midterm Project Plan Presentation Template) - 複製.docx`
- `W11：最終專題進度報告樣板(Final Proposal Progress Presentation Template) - 複製.docx`
- `W15-16：期末展示_最終繳交內容樣板(Final Demo _ Submission Checklist).docx`

### 2. Markdown 編輯版

用來整理內容、編輯草稿與持續撰寫的版本。

- `W8：期中專題企畫報告.md`

### 3. 最終繳交版

用來保存最後要交出的正式版本。

- `W8：期中專題企畫報告.docx`

## 使用方式建議

- 先參考老師提供的 `.docx` 樣板確認格式與要求
- 實際撰寫與整理內容時，以 `.md` 檔為主
- 完成後再手動整理成最終繳交的 `.docx` 版本
